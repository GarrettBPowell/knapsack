Contents: Knapsack Solver Algs
Description: 
This project is a collection of different knapsack solver algorithms. It contains 3 greedy algs, exhaustive, exhaustive with pruning, and a simmulated annealing algorithm.

How to run:
This is a visual studio solution. 
main.py is the driver script. 
All test cases are in the resources folder and can be called by just typing their name without the extension when prompted.
The menu has all single alg options with a few multi alg combination options.
